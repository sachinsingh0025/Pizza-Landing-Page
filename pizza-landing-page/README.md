# 🍕 Pizza. | Premium Pizza Landing Page & Ordering System

Welcome to **Pizza.** – A high-end, interactive web application for pizza lovers. This project features a modern, mobile-responsive "Glassmorphism" design with a smooth user experience from landing to order tracking.

![Pizza. Preview](https://images.unsplash.com/photo-1513104890138-7c749659a591?auto=format&fit=crop&w=800&q=80)

## ✨ Features

*   **💎 Modern UI**: Premium aesthetics using Glassmorphism, smooth CSS animations, and Poppins typography.
*   **🍕 Interactive Menu**: Filterable menu (Veg, Non-Veg, Spicy) with dynamic grid rendering.
*   **🛠️ Custom Pizza Builder**: Build your own masterpiece! Pick toppings and watch the price update in real-time with visual feedback.
*   **🛒 Sleek Cart Sidebar**: Manage your items in a sliding cart panel without leaving the page.
*   **💳 Secure Checkout Flow**: A multi-step mobile-style payment interface with card validation and success/failure states.
*   **🚚 Real-time Order Tracking**: Animated delivery tracking page with a simulated map view and progress stepper.
*   **📱 Fully Responsive**: Optimized for Desktop, Tablet, and Mobile devices.
